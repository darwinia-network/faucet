# @darwinia/dactle

Darwinia telegram bot

## Usage

### Use environments

```bash
export TELEGRAM_BOT_TOKEN=19847361:48348abde
export DACTLE_SEED=0x93827abcdef
./dactle.sh bot
```

### Use command

```bash
./dactle.sh bot [TOKEN]
```
